About Me

I am...

Filip Jęczmyk. I started my adventure with programming mobile applications relatively recently - 6 months ago. Projects that I created are not complicated, but what I care most about is that they are done carefully. The code is transparent, thanks to the use of object-oriented programming, and the application visually catches the eye of the potential user. In near future I want to develop and create even better and more refined applications.



All projects were programmed in Kotlin and using Jetpack Compose software. Technologies that I used to create these projects include: ROOM, Navigation, Dagger.



I have been programming for 3 years and it is my passion and everyday life. 2D and 3D graphics are also familiar to me. I am also familiar with languages such as: C #, Python, Javascript, CSS and the HTML markup language. I am looking for a job in which I can learn and develop a lot, in which I can learn both team and individual work.
